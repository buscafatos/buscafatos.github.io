#fact checking | #anti fake-news

Fazer checagem de fatos, afirmar que uma notícia é, ou não, fake-News tornou-se uma tarefa complexa. E fazer isso, de forma automatizada, é um trabalho desafiador.

Brasil é um país aonde fake-news (e redes sociais) se tornaram um problema para a democracia. Com isso em mente, resolvemos humildemente criar bots e ferramentas de código aberto para ajudar na checagem de fatos e no combate as fake-news, sempre tentando seguir códigos de príncípio pré estabelecidas pelo IFCN. No Brasil existem poucos sites com o selo IFCN e estamos trabalhando para conseguir também para o Busca Fatos.

Sobre o coletivo

Somos um pequeno grupo de cidadãos brasileiros insatisfeito com a quantidade de desinformação e fake-news que é compartilhada nas redes sociais, principalmente WhatsApp. Acreditamos que a tecnologia pode e deve contribuir para que essa pesquisa torne-se mais fácil para todos.

Parceiros

Utilizamos a QODDI em função de essa empresa promover um programa de incentivo ao código aberto.

Conheça nossas ferramentas

Criamos uma API e disponibilizamos online. Ela possui pouca inteligência, vamos melhorando ela nas próximas versões - e para facilitar criamos algumas interfaces de consumo da API.

Whats app BOT +55 61 8165-7213

Um simples bot para whatsApp com o número +55 61 8165-7213. Ele chama a API e retorna os resultados mais relevantes. Estamos constantemente melhorando e incluindo funcionalidades.

Interface WEB

Criamos uma interface minimalista para ajudar-nos a testar e melhorar a busca realizada pela nossa API. Ela ordena por data e acaba sendo mais fácil visualmente de usar do que os bots.

Telegram BOT

Mesma coisa que o bot para whatsApp - criando mais opções de checagem da informação. Ele também funciona em grupos - basta adicionar ele no grupo e dar reply em uma mensagem marcando @buscafatos_bot.
